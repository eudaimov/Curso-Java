Argumentos Variables en los métodos

Se utilizar para pasar argumentos a una función o constructor y desconzco la cantidad de argumentos los cuales voy a pasar.

Por ejemplo:

public void metodo(Object ... varArgs){ Object objecto = varArgs[0]; } public void metodo2(int ... varArgs){ int cantidad = varArgs.lenght; int numero = varArgs[0]; }

Otro ejemplo un poco más extenso:

public class argumentosVariables { public static void main(String[] args) { int[] numeros = {24,7,18,96,87,45}; metodoArgumentosVariables(numeros[0], numeros[3], numeros[1]); } public static void metodoArgumentosVariables(int ... args){ int cantidad = args.length; System.out.println("La cantidad de argumentos pasados son "+cantidad); System.out.println("Y sus valores son"); for (int valorNumeroArgumento: args ) { System.out.println(valorNumeroArgumento); } } }
