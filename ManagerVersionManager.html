Introducción

Existen varios programas que se encargan de administrar la instalación de las diferentes versiones JDK y conviene conocer al menos uno. En nuestro caso nos encargaremos de SDKMAN.

Solo funciona en windows con PowerShell 7 no funciona con el cmd

Instalación en Windows.

En nuestra terminal de powerShell escribimos lo siguiente:

Install-Module -Name posh-sdk

Mirar lo siguiente:

notepad $PROFILE // y debe aparecer: Import-Module Posh-SDKMan

Si te da error sdk version prueba a crear un archivo VERSION.txt con el siguiente contenido 2.0.0 en la carpeta: C:\Users\[Usuario]\Documents\PowerShell\Modules\Posh-SDKMan\2.0.0

Uso:

  • sdk version: Para ver la version:
    posh-sdk (POwer SHell Groovy enVironment Manager) 2.0.0 base on SDKMAN! 1.3.13 and SDKMAN! API 5.18.2
  • sdk help: Para ver la ayuda.
  • sdk offline enable|disable
  • sdk list java | maven | gradle | ant Para listar versiones de sdk disponible

    • Para un listado completo visitar: SDK's

  • sdk install java )o cualquier otra sdk) version Para instalar.
  • sdk use java 8.0.422-zulu Para cambiar y utilizar otra sdk.
  • sdk selfupdate force Para actualizar los listados
  • sdk current java. La version que se esta utilizando.
