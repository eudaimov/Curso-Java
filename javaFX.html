JavaFX

Definición

Es la plataforma de código abierto para crear aplicaciones de escritorio o móviles de Java

La base

La base de cualquier aplicación JavaFX son:

  • Stage
  • Scene
  • Node

Hay tres tipos de Nodo: Raiz, Rama y Hoja.

Un código base del que partir sería:

public class HelloApp extends Application { private Parent createContent() { return new StackPane(new Text("Hello World")); } @Override public void start(Stage stage) throws Exception { stage.setScene(new Scene(createContent(), 300, 300)); stage.show(); } public static void main(String[] args) { launch(args); } }
