Logger

Introducción

Los logger son una herramienta en Java que se utilizar para registrar mensajes de una aplicación Estos mensajes pueden ser útiles para depuración de errores, monitoreo y análisis de comportamiento del programa.

La clase principal en Java sin necesidad de instalar ninguna dependencia es Java.Util.Logging.Logger.

Los principales librerías de logs son SLF4J, Log4j2 de Apache, LogBack y JBossLoggin.

Niveles de Log

Existen diferentes niveles de Logs y dependen de la librería que estes utilizando.

Principales Librerias de Logs:

Log4j

Descripción: Una de las librerías de logging más antiguas y populares. Ofrece una gran flexibilidad y es altamente configurable.

Características: Soporte para múltiples niveles de log, configuración mediante archivos XML o propiedades, y capacidad para log rotativo.

Ejemplo:

			
				import org.apache.log4j.Logger;

				public class EjemploLog4j {
				    private static final Logger logger = Logger.getLogger(EjemploLog4j.class);
				
				    public static void main(String[] args) {
				        logger.info("Mensaje informativo");
				        logger.error("Mensaje de error");
				    }
				}

SLF4J (Simple Logging Facade for Java):

Descripción: No es una librería de logging en sí, sino una fachada que permite a los desarrolladores utilizar diferentes librerías de logging (como Log4j, Logback) sin cambiar el código de la aplicación.

Características: Abstracción sobre varias implementaciones de logging, lo que facilita cambiar de una librería a otra.

Ejemplo:


			import org.slf4j.Logger;
			import org.slf4j.LoggerFactory;
			
			public class EjemploSLF4J {
			    private static final Logger logger = LoggerFactory.getLogger(EjemploSLF4J.class);
			
			    public static void main(String[] args) {
			        logger.info("Mensaje informativo");
			        logger.error("Mensaje de error");
			    }
			}

Logback:

Descripción: Desarrollada por el creador de Log4j, es considerada su sucesora. Ofrece mejor rendimiento y es más moderna.

Características: Configuración mediante archivos XML o Groovy, soporte para log rotativo y filtrado avanzado.

Ejemplo:


			import org.slf4j.Logger;
			import org.slf4j.LoggerFactory;
			
			public class EjemploLogback {
			    private static final Logger logger = LoggerFactory.getLogger(EjemploLogback.class);
			
			    public static void main(String[] args) {
			        logger.info("Mensaje informativo");
			        logger.error("Mensaje de error");
			    }
			}

Log4j2

Descripción: La versión mejorada de Log4j, con mejoras en rendimiento y características adicionales.

Características: Mejor rendimiento que Log4j, configuración flexible, y soporte para JSON, XML y YAML.
