instanceOf

¿Qué es?

Es un operador binario que usamos para probar si un objeto es de un tipo determinado. El resultado de la operación es verdadero o falso . También se le conoce como operador de comparación de tipos porque compara la instancia con el tipo.

Antes de lanzar un objeto desconocido, siempre se debe utilizar la instancia de verificación. Hacer esto ayuda a evitar una ClassCastException en tiempo de ejecución.

La sintaxis básica del operador de instancia es:

(intancia) instanceOf (type/class)

¿Cómo funciona la instancia del operador?

Este operador sólo dará falso en el caso de que la instancia sea nula, pero si la instancia y el tipo no son de la misma clase o no implementa la interfaz o no extiende de esa clase el resultado será un error en tiempo de ejecución:

java: incompatible types: XXX cannot be converted to XXX

Hay otro caso en el que también se puede utilizar que es aquellos que estamos utilizando una instancia de una clase padre para compararla con una de la clase hija. En este caso dara falso.
