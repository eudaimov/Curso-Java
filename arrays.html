Arreglos

Definición

Es una colección de variables del mismo tipo a las que se hace referencia por medio de un nombre común.

La principal ventaja de un arreglos es que organiza los datos para facilitar su manipulación

Creación de arreglos

Tenemos que saber antes de crear la capacidad de elementos que puede contener el array. Esta capacidad o tamaño es inalterable y fija.

int [] arrayNumero = new int[5]; int arrayNumero[] = {1,2,3,4,5};

Para introducir datos en una posición determinada de un array

int arrayNumero[] = new int[5]; arrayNumero[0]=1; arrayNumero[1]=2; arrayNumero[2]=3; arrayNumero[3]=4; arrayNumero[4]=5;

La primera posición de cualquier array siempre comienza con 0

Recorrer posiciones de un array

Hay dos maneras a traves de un for y un foreach:

for(int i = =0; i < arrayNumero.lenght;i++){ System.out.println(arrayNumero[i]); }

foreach:

for(int element: arrayNumero){ System.out.println(element) }

¿Cómo copiar un array en otro?

Para copiar un primer array dado en un segundo array, hay que poner el array primero que deseo copiar y desde la posición que deseo copiar, luego el array de destino y la posición en la que deseo empezar a ubicar el array de destino, y por último la cantidad de elementos que deseo copiar.

int[] arrayPrimero = {10,20,40,101,25}; int[] arraySegundo = new int[5]; System.arraycopy(arrayPrimero, 0, arraySegundo, 0, arrayPrimero.lenght);
