Expresiones Lambda

Las expresiones lambda en Java son una característica introducida en Java 8 que te permite crear funciones anónimas. Estas funciones se pueden usar en línea, sin necesidad de definirlas de forma tradicional con un nombre y un cuerpo.

Ejemplo:

Supongamos que queremos ordenar una lista de números de mayor a menor. Sin usar expresiones lambda, tendríamos que crear una clase que implementa la interfaz Comparator y definir un método compare que compare dos números:

class NumeroComparator implements Comparator <Integer> { @Override public int compare(Integer num1, Integer num2) { return num2 - num1; } } List <Integer> numeros = Arrays.asList(1, 5, 3, 2, 4); numeros.sort(new NumeroComparator());

Con las expresiones lambda, podemos escribir la misma lógica de comparación de forma más concisa:

List<Integer> numeros = Arrays.asList(1, 5, 3, 2, 4); numeros.sort((num1, num2) -> num2 - num1);

En este ejemplo, la expresión lambda (num1, num2) -> num2 - num1 define una función que toma dos números como entrada y devuelve la diferencia entre ellos. Esta función se utiliza como argumento del método sort para ordenar la lista de números de mayor a menor.

Beneficios de usar expresiones lambda:

  • Código más conciso: Las expresiones lambda te permiten escribir código más corto y fácil de leer.
  • Mejora la legibilidad del código: El código con expresiones lambda es más expresivo y fácil de entender.
  • Simplifica el código funcional: Las expresiones lambda son una herramienta poderosa para escribir código funcional en Java.
